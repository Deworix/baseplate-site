The flagship loop

Playtest → error → fix → PASS.
Without you in the middle.

Every Roblox dev knows the ritual: playtest, squint at the Output window, alt-tab, guess, repeat. Baseplate's bridge gives agents eyes inside Studio — they run the playtest, read the actual runtime error, fix the cause in your source, and re-run until the console is clean.

01

Playtest

The agent runs your game through the live Studio bridge and watches the console — the same output you'd read, minus the squinting.

02

Diagnose

It reads the real stack trace, finds the cause in your Luau source, and explains it in plain language — not a cryptic dump.

03

Fix

It edits src/ directly. Rojo syncs the change into Studio the moment it's written.

04

Verify

It playtests again and only reports PASS when the console is actually clean. No "should work now".

Exploit Guard

Every unvalidated RemoteEvent is free money for exploiters.

If your server trusts the client — prices, rewards, damage — someone will find it. Exploit Guard audits your live game like an attacker would: it maps every remote, tells you each hole as a plain-language attack story ranked by severity, and writes the server-side hardening for you.

  • Finds remotes that trust the client before players do
  • Explains each exploit as the attack it enables — "a client can dictate its own price"
  • Writes validated, server-authoritative fixes into your source

Included free Exploit Guard ships in the Free plan. Protecting your revenue shouldn't be a paywall.

FAQ

Fair questions.

Do I need my own AI subscription?

Yes — Baseplate drives your existing agent CLI account. Claude Code, Codex and Cursor support the verified Playtest Copilot and can be mixed in one Crew; Grok is available as a normal builder tile. Baseplate does not resell AI tokens.

Does my code leave my machine?

Baseplate does not upload your projects or keys to Baseplate. Your selected AI provider receives prompts, code and context needed for the work. License validation sends your key and a random installation identifier to our Cloudflare service. If you enable team collaboration, board cards, shared prompts and display names sync through the team service; source files and API keys do not.

How is this different from just using Claude Code in a terminal?

The bridge. A terminal agent can write Luau, but it's blind — it can't playtest, can't read the Output window, can't check the live DataModel. Baseplate gives agents eyes and hands inside Studio, plus the deck, board, swarms and one-click missions built on top.

How do I cancel or change my plan?

Anytime, yourself — no email required. Open the billing portal and sign in with the email you paid with; from there you can cancel, switch between Pro and Studio, update your card, or download invoices. A cancellation keeps your plan active until the end of the period you've already paid for, then drops you to Free. If the portal link isn't working yet, email support@baseplatedev.com and we'll sort it out.

What happens when my Pro subscription lapses?

You drop back to Free — one agent at a time, with the bridge, Regression Lab and Remote Contract Compiler still included. Your projects, workspaces and history stay untouched on your disk.

Is the Studio plugin safe?

The plugin only talks to Baseplate on 127.0.0.1. The local server rejects foreign Host and browser Origin values, the Electron renderer is sandboxed, and release packages use ASAR integrity checks. No security control can guarantee invulnerability, so review agent-generated changes before you ship.

I'm a student / hobbyist. Is Free actually usable?

Free is the full core loop: one agent with complete Studio access, Playtest Copilot, Rojo sync, Regression Lab and Remote Contract Compiler. It's deliberately generous — if Baseplate never earns your upgrade, it should still make your game better.
